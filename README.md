# Aromas and Cuidado

This is a web application for showcasing and selling aromatic and skincare products.

## Features
- Product catalog
- Team section
- Modern UI with Tailwind CSS

## Getting Started
1. Install dependencies:
   ```
   npm install
   ```
2. Start the development server:
   ```
   npm run dev
   ```

## Folder Structure
- `src/` - Source code
- `public/` - Static assets
